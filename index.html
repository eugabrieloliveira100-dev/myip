How to check IP via Terminal

Windows (CMD / PowerShell)

Check your local network configuration:

> ipconfig

Linux & macOS

List all network interfaces and addresses:

$ ip a # Modern Linux
$ ifconfig # macOS/Legacy

Pro Tip: To see your Public IP (this website's result) in terminal, use: curl ifconfig.me
